# CI_OS Package Repository

Все пакеты и БД хранятся в GitHub Release (без ограничений размера).

```ini
[ci_os]
SigLevel = Optional TrustAll
Server = https://github.com/s3t02sp/ci_os-repo/releases/download/packages-latest
```
